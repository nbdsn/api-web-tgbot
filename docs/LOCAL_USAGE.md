# 本地使用说明

## 1. 启动

```bash
go mod tidy
go run .
```

访问：`http://127.0.0.1:8088`

默认账号密码：

- 账号：`admin`
- 密码：`admin`

## 2. 首次配置

1. 进入“主程序连接”页面
2. 填写主程序地址、管理员账号、密码
3. 如需数据库模式，开启后再填写数据库路径（支持一键搜索）
4. 可选配置代理（支持 `http://` / `socks5://`）
5. 点击“保存连接配置”
6. 点击“验证可用性”（会校验 status + user + channel 接口）

## 3. 配置 TG

1. 进入“TG 功能”
2. 设置启用开关、Token、管理员 ID（逗号分割）
3. 设置额度换算比例（默认 100=50000000）
4. 保存
5. 点击“发送测试消息”验证 TG 通道

## 4. 自动额度处理

1. 打开“自动额度处理”页面
2. 设置执行时间、低/高阈值与目标值、白名单、是否推送管理员报告
3. 保存配置
4. 可点击“立即执行一次”手动触发
5. 在日志表查看处理记录

## 5. 账号安全

进入“账号设置”修改默认账号密码。
