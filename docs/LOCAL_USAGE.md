# 本地使用说明

## 1. 启动

```bash
go mod tidy
go run .
```

访问：`http://127.0.0.1:8088`

默认账号密码：

- 账号：`admin`
- 密码：`admin`

## 2. 首次配置

1. 进入“主程序连接”页面
2. 填写主程序地址、管理员账号、密码
3. 填写主程序数据库路径（可点击“一键搜索”）
4. 点击“保存连接配置”
5. 点击“测试连接”确认可联通

## 3. 配置 TG

1. 进入“TG 功能”
2. 设置启用开关、Token、管理员 ID（逗号分割）
3. 设置额度换算比例（默认 100=50000000）
4. 保存
5. 在 TG 对话中输入 `/start` 检查命令菜单

## 4. 渠道管理

- 新增渠道：填写名称、类型、Key、状态后点击新增
- 列表刷新：点击刷新列表
- 行内操作：启停或删除

## 5. 账号安全

进入“账号设置”改掉默认账号密码。
